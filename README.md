## Description

This microservice is responsible for integrating with Bento.ky to receive delivery fees and apply a new margin value to calculate a new fee. All requests are saved in the database, allowing users to query and consult all calculations.

## Project setup

```bash
$ yarn install
$ docker compose up -d
$ npm run migration:run
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
